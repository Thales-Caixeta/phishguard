# PhishGuard IA 🛡️

Sistema inteligente de análise e prevenção de fraudes digitais desenvolvido com IA heurística.

## Sobre o Projeto

O PhishGuard IA permite que qualquer usuário cole uma mensagem ou link suspeito e receba em segundos uma análise de risco com nível de ameaça (Baixo, Médio ou Alto), elementos suspeitos detectados e recomendações de segurança.

Desenvolvido como projeto acadêmico para a disciplina de Inteligência Artificial.
